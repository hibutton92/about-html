TAG : < > ... < / > ,설명하기

  • - br : html new line tag ,줄바꿈
  • - p , /p : html paragragh tag ,단락
  • - img src="" width="100%" : 이미지 삽입, 넓이 설정
  • - li , /li : 리스트 작성 (자식태그)
  • - ul , /ul : Unordered list, 여러 li를 구분하는 역할(부모태그)
  • - ol , /ol : ordered list, 여러 li를 구분하는 역할, 숫자가 매겨짐(부모태그)
  • - title , /title : 웹페이지의 제목을 사용자에게 명시적으로 알려줌, 검색엔진에 노출, 책으로 치면 책표지
  • - meta charset="utf-8" : "웹브라우져야, 이 웹을 열때는 'UTF-8'로 열렴!"
  • - body , /body : 본문에 대한 내용이라는 약속
  • - head , /head : 본문을 설명하는 태그라고 약속
  • - html , /html : 헤드와 바디를 감싸는 태그
  • - !doctype html : "이 페이지는 html이다"
  • - a , /a : anchor, 링크하기
